## 📌 Project Overview

This repository contains the practical implementation of core cryptographic concepts covered in **BIT4138 Advanced Cryptography** (Weeks 1–6). The project demonstrates:

- Classical ciphers (Caesar, Vigenère)
- Stream ciphers (LFSR, RC4) & randomness testing
- AES block cipher (CBC mode, PBKDF2, file encryption)
- RSA public‑key cryptography (key generation, encryption/decryption)
- Password hashing with bcrypt + SHA‑256 and login authentication

All code is written in **Python 3** using the `cryptography` library, `bcrypt`, and standard libraries. The logbook (screenshots & reflections) is maintained separately as a Word/PDF document.
